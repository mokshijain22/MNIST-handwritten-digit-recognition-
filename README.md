# MNIST Digit Recognizer

A CNN-based handwritten digit recognizer built with TensorFlow and Flask.

## Live Demo
https://mnist-handwritten-digit-recognition.onrender.com

## Features
- Draw digits on canvas and get real-time predictions
- Upload images for prediction
- CNN model with 99%+ accuracy trained on 60,000 MNIST samples

## Tech Stack
- Python, TensorFlow, Keras, Flask, HTML/CSS/JS

## Run Locally
```bash
pip install -r requirements.txt
python app.py
```
Then open http://127.0.0.1:5000
